# Computer_Networks_Mini-Project
A network proposal has to be developed for a bank. The bank has a main office, which is located in Chennai, and has 5 branch offices located at Coimbatore, Madurai, Trichy, Salem and Thirunelveli. The bank has an application server, which is used by its customers across the world for online transactions. All the branches have high speed internet connection. There are approximately 50 users in each of the branch offices and 100 users in the main office.
